# shopifyBaseTheme
a shopify 2.0 base theme that passes `shopify theme check`


# CLI Commands
`shopify login --store yoursitename.myshopify.com`

`shopify logout`

`shopify theme check`

[Source](https://shopify.dev/changelog/online-store-2-0-detect-theme-errors-with-theme-check)

`shopify theme serve`
